# Garden мини-игра — Спецификация

## Обзор

Сад — Telegram Mini App внутри бота Trashkek. Игрок ухаживает за грядками,
сажает семена, собирает урожай, открывает подвиды, торгует на рынке.

### Реализованные механики

#### P0-A: Автопродажа подсолнуха

**Где**: `manager.ts`, метод `getAutoSellPrice()`

- Автопродажа работает **только для подсолнухов** (plant_id=1)
- Формула:

```
autoSellPrice = Math.round(50 × subspecies.price_mult × 1.01)
```

- Для базового (common) `price_mult=1.0`: **51🪙** (прибыль 1🪙 ≈ 1%)
- Для всех остальных растений автопродажи нет

**Таблица прибыли подсолнухов:**

| Подвид | Семя🪙 | Авто🪙 | Прибыль | Рост |
|--------|:-----:|:------:|:-------:|:----:|
| Обычный (common) | 50 | **51** | **+1** | 20 мин |
| 🔴 Красный (uncommon) | — | **61** | **+11** | 18 мин |
| 🟠 Оранжевый (rare) | — | **76** | **+26** | 16 мин |
| 🟡 Махровый (epic) | — | **101** | **+51** | 14 мин |

Семена подвидов не продаются в магазине — их цена на рынке устанавливается
игроками.

#### P0-B: Кривая открытия подвидов (Discovery)

**Где**: `manager.ts`, метод `upgradeItem()`

**Формула**:

```
открыто = число записей в garden_discoveries WHERE plant_id = X AND user_id = Y
всего  = число подвидов WHERE plant_id = X (включая базовый)

шанс = 25% - (открыто / всего) × 15%
```

| Открыто подвидов | Шанс |
|:----------------:|:----:|
| 0 из 4 | 25% |
| 1 из 4 | 21.25% |
| 2 из 4 | 17.5% |
| 3 из 4 | 13.75% |
| 4 из 4 | — (стоп) |

**Правила:**
- Проверка `открыто < всего` до списания 💎 (ошибка `"Все подвиды этого растения уже открыты"`)
- При успехе: weighted random по `discover_weight` среди неоткрытых
- Запись в `garden_discoveries` + выдача 1× семени подвида в инвентарь
- Весь апгрейд (level up + discovery) — в одной `BEGIN IMMEDIATE` транзакции

#### P0-C: Рынок (только 🪙) + комиссия

**Где**: `manager.ts`, методы `buyFromMarket()`, `sellOnMarket()`, `cancelTrade()`

- Валюта: **только 🪙** (колонка `currency` удалена)
- 💎 используется только для улучшений и покупки грядок
- **Комиссия**: 5% с продавца, сжигается (TX_TYPE=7)
  - `sellerPayout = price - Math.ceil(price × 0.05)`
- **Минимальная цена лота**: 10🪙

**Паттерн покупки (защита от race condition):**

```
1. BEGIN IMMEDIATE
2. SELECT status FROM garden_trades — проверка 'active'
3. Если не active → throw Error("Лот уже куплен")
4. withdraw(price) у покупателя
5. deposit(sellerPayout) продавцу, deposit(fee, банк=0) сжигание
6. UPDATE inventory SET user_id=new, tradeable=1
7. UPDATE trades SET status='sold'
8. COMMIT
```

#### P0-D: Всхожесть и обломки (Shards)

**Где**: `manager.ts`, метод `plantSeed()`

При посадке — бросок всхожести `Math.random() < subspecies.germination_chance`:
- **Успех**: `state='growing'`, запись `subspecies_id`, `planted_at=NOW`
- **Провал**: `state='withered'`, `withered_at=NOW`, обломки:

| Редкость семени | Обломков |
|:--------------:|:--------:|
| common | 1 |
| uncommon | 2 |
| rare | 3 |
| epic | 4 |
| legendary | 5 |

**Обломки (shard):**
- `type = 'shard'`, `plant_id = NULL`, `subspecies_id = NULL`
- Один стак на пользователя, `quantity` до 999
- **Крафт**: 5 обломков → 1 любое базовое семя на выбор

**UI**: кнопка **"🔨 Крафт (N/5)"** в инвентаре (shard-фильтр),
при >= 5 — модалка с выбором растения.

#### P0-E: Lazy-очистка сгнивших грядок

**Где**: `queries.ts`, метод `plots.all()`

```
if plot.state == 'withered' AND (сейчас - withered_at) > 15 секунд:
    очистить: state = 'empty', plant_id = NULL, subspecies_id = NULL,
              planted_at = NULL, stage = 0, withered_at = NULL
```

Фронтенд: показывает 🥀 на ~15 секунд с обратным таймером (красный цвет).
После очистки — пустая грядка.

## Полный игровой цикл

1. `/daily` (награды сада — **не реализовано**)
2. `/garden` → mini-app
3. **Магазин**: купить базовые семена за 🪙 (лимит 100/запрос)
4. **Посадка**: пустая грядка → seed → бросок всхожести
   - Успех: `growing`, таймер
   - Провал: `withered` 🥀 15с → empty, +1-5 обломков
5. **Ожидание**: 20–180 мин
6. **Сбор**: подсолнух → автопродажа **или** в инвентарь; остальное → только в инвентарь
7. **Инвентарь**: продать (рынок / автопродажа), улучшить 💎, скрафтить обломки
8. **Улучшение**: 💎 → level +1 + шанс открыть подвид
9. **Рынок**: выставить / купить / отменить (только 🪙, 5% комиссия)

## Транзакции

Все мутирующие функции — `BEGIN IMMEDIATE`:

| Функция | Операции |
|---------|----------|
| `plantSeed()` | Проверка грядки → germ roll → списание предмета → growing или withered + shard |
| `upgradeItem()` | Проверка `открыто < всего` → списание 💎 → level up → discovery roll |
| `buyFromMarket()` | `SELECT status` → withdraw → deposit/payout/fee → transfer → markSold |
| `buyPlot()` | Проверка `plots_max < 9` → списание 💎 → `UPDATE plots_max += 1` |
| `craftShards()` | Проверка `quantity >= 5` → списание 5 → выдача 1 seed |
